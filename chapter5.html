Chapter 5 : Stage และ Transition

ในส่วนของบทนี้จะเป็นการอธิบายในส่วนของหน้ากระดาษที่ใช้ทำงานอย่างตัว Stage ซึ่งเป็นพื้นที่สีขาวที่เมื่อเราสร้าง File งานขึ้นมาเราจะเจอเป็นหน้าแรก ซึ่งหน้า Stage นี้จะใช้สำหรับสร้างชิ้นงานและแสดงผลงานที่เราได้สร้างขึ้นมา และจะอธิบายถึงการปรับวัตถุหรือ Instance ต่างๆ อย่าง Transition ไมว่าจะเป็นการปรับขนาด ย่อขยาย เคลื่อนย้าย หรือแม้แต่การปรับจุดหมุนของวัตถุหรือ Instance นั้นๆ และยังมีตัวอย่างการสร้างตัวอย่าง Animation 2 มิติ เล็กๆน้อยๆให้ได้ชมกันอีกด้วย ซึ่งสามารถรับชมกันต่อได้ที่ช่อง Animashort
