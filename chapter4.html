Chapter 4 :Layer, Symbol และ Instance และ Library Panel

ในส่วนของบทนี้เราจะมาพูดถึงตัวของ Layer, Symbol และ Library ซึ่งใน 3 ส่วนนี้ นั้นมีผลในการสร้าง Animation เป็นอย่างมาก โดย Layer คือ ส่วนหนึ่งของโปรแกรมที่มีความสำคัญกับการสร้าง Animation และควบคุมการเคลื่อนไหว ซึ่ง Layer จะทำหน้าที่เปรียบเสมือนแผ่นใส ใช้วางวัตถุซ้อนทับกัน โดยแต่ละ Layer ก็จะแยกเป็นอิสระต่อกัน แต่จะประกอบกันเป็นงานชิ้นเดียวได้ และ Symbol คือการนำวัตถุที่เป็นภาพ Graphic, Button หรือ Movie Clip มากำหนดเป็นต้นแบบ เพื่อนำไปใช้เป็นส่วนประกอบในชิ้นงาน ซึ่งจะทำให้การสร้าง Animation นั้นเป็นไปอย่างง่ายดาย เมื่อต้องสร้างภาพเดิมหลายๆภาพ โดย Symbol ที่สร้างจะถูกเก็บไว้ใน Panel “Library” ซึ่ง Panel นี้ จะทำหน้าที่รวม Symbol ทั้งหมดที่เราสร้างขึ้นมาไว้ที่ Panel นี้ และจะแสดงภาพ Preview เมื่อเราคลิกไปที่ Symbol ที่เราเลือก และจะแสดง Symbol ที่เราสร้างไว้ ซึ่ง Library Panel เมื่อเราต้องการนำ Symbol มาใช้ ให้เราเลือก Symbol ที่ต้องการและลากมาวางที่หน้ากระดาษของเราได้เลย ซึ่งสามารถทำซ้ำได้เรื่อยๆ โดยไม่มีผลกับภาพต้นฉบับจนกว่าเราจะเข้าไปแก้ไขใน Symbol เสียเอง ถ้าหากต้องการรับความรู้เพิ่มเติมสามารถรับชมเนื้อหาเพิ่มเติมได้ที่ช่อง Animashort
